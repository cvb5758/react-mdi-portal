# mdi-portal-lsm-study

MDI 포털 풀스택 구현 스터디

## 1. 개발환경

- ### FE : Vanila JavaScript, HTML, CSS
- ### BE : Node, Express, MySQL
- ### 버전 관리 : Github
- ### 배포 환경 : AWS
