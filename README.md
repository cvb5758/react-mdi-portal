# mdi-portal-lsm-study

MDI 포털 풀스택 구현 스터디

## 1. 개발환경

- ### FE : Vanila JavaScript, HTML, CSS
- ### BE : Node, Express, MySQL
- ### 버전 관리 : Github
- ### 배포 환경 : AWS

## 2. 프로젝트 구조

```
├── README.md
├── LICENSE
├── docs
│    └── daily.md
└── src
```

## 기능과 UI

- ### 기능

  - [ ] 회원가입, 로그인, 로그아웃
  - [ ] 새로운 작업(task) 생성
  - [ ] 저장(로그인 상태)
  - [ ] 작업 Drag & Drop
  - [ ] 작업 숨기기 및 최대화
  - [ ] 진행중인 작업 정렬
  - [ ] 모든 작업 열기 및 최소화

- ### UI
  - [ ] 회원가입 폼
  - [ ] 로그인 폼
  - [ ] 새로운 작업 생성 폼
  - [ ] 상단 header에 제목 표시
  - [ ] header 우측에 닉네임(로그인 상태) 표시
  - [ ] footer에 진행 작업 표시
  - [ ] footer에 nav 활성화 버튼
